How to use the app

Launch:
	Open terminal and navigate inside the Sales_project folder and run the following command
	
	streamlit run app.py

----------------------------------
Inside the app

Filters: 

	Select the cluster you wish to visualize
	Select the Item type you wish to visualize

----------------------------------

Sales Segmentation Dashboard:
	
	Total SKUs: Stock Keeping Unit (total item) in your selected filters
	Avg retail sales
	Avg warehouse sales
	
----------------------------------
	
Item Lookup:
	
	Enter your Item code to see the name of the product and Cluster it falls under
	
----------------------------------

Cluster Sales Profile: 
	
	Visualize Retail and Warehouse sales within each selected clusters
	
----------------------------------

Cluster Interpretation:

	Describes each cluster in simple form
	
----------------------------------

Item Type Contribution per Cluster:

	How much share does each item type take inside a cluster
	
----------------------------------

Monthly Sales Trend:

	Select between warehouse and Retail sales and visualize the monthly sales pattern in each cluster
